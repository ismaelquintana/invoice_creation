  - amount: pos_integer()

  - next_id: non_neg_integer()

API SUMMARY
================================================================================

InvoiceStorage (Main Module):
  save/1              Save single invoice
  load/1              Load single invoice
  delete/1            Delete invoice file
  exists?/1           Check if invoice exists
  save_all/1          Save multiple invoices
  load_all/1          Load all invoices for a year
  save_year_list/1    Save ListInvoiceYear
  load_year_list/1    Load ListInvoiceYear
  list_years/0        List all years with invoices
  count/1             Count invoices in a year

InvoiceStorage.Encoder (Serialization):
  encode_invoice/1    Invoice -> JSON String
  encode_list_invoice_year/1  ListInvoiceYear -> JSON String
  encode_item/1       Item -> JSON String

InvoiceStorage.Decoder (Deserialization):
  decode_invoice/1    JSON String -> Invoice
  decode_list_invoice_year/1  JSON String -> ListInvoiceYear
  decode_item/1       JSON String -> Item

InvoiceStorage.Error (Error Type):
  file_not_found/1    File doesn't exist
  permission_denied/1 Access denied
  disk_full/1         No space to write
  encode_failed/1     Encoding failed
  decode_failed/2     Decoding failed
  invalid_json/1      JSON syntax error
  invalid_invoice_data/1  Validation failed
  invalid_year/1      Year validation failed
  io_error/2          Generic I/O error
  to_user_message/1   Format for user display

ERROR HANDLING OVERVIEW
================================================================================

Error Type Classification:

RECOVERABLE ERRORS (Can retry):
  - :file_not_found    -> Create new or retry load
  - :disk_full         -> Free space and retry
  - :io_error          -> Retry with backoff

PERSISTENT ERRORS (Need intervention):
  - :invalid_json      -> Corrupted file, manual fix
  - :invalid_invoice_data -> Schema mismatch, migration
  - :permission_denied -> Fix permissions
  - :invalid_path      -> Fix code bug
  - :encode_failed     -> Debug and fix
  - :decode_failed     -> Investigate corruption

All errors include:
  - Type classification (:file_not_found, etc)
  - Human-readable message
  - Contextual information (paths, reasons, etc)
